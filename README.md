# CLANITYproject
[원데이 클래스 + 커뮤니티] 
내 주변 취미 공유 및 One-Day Class 추천 커뮤니티 플랫폼

## 프로젝트 소개
취미를 공유하고 소통하는 커뮤니티형 원데이 클래스 플랫폼
- 강사와 수강생 간 교류, 수강생 간 커뮤니티 기능 제공
- 일반 사용자간의 자유로운 커뮤니티 기능
